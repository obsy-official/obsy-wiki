# Как установить плагины 🔌

## 🤔 Как это работает?
Плагины обычно ставятся в папку `plugins` на выключенном сервере.

---

## 🚀 Как установить плагин?
1. Остановите сервер
2. Скачайте нужный `.jar` файл плагина
3. Откройте папку `plugins`
4. Загрузите туда файл плагина
5. Запустите сервер
6. Проверьте логи на наличие ошибок

---

## 💡 Полезный совет
Если плагин не запускается:
- проверьте версию сервера
- проверьте зависимые плагины
- посмотрите ошибки в консоли

Эта статья полезна, если Вы ставите, например, **SkinsRestorer** для [отображения скинов](/sections/faq/skins).
