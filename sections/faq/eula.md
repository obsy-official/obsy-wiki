# Лицензионное соглашение 🤝

## 🤔 Что такое EULA?
**EULA** (End User License Agreement) — это правила использования игры и серверного ПО.

По умолчанию сервер не запустится, пока Вы не подтвердите, что согласны с этими правилами 🛑

---

## 🚀 Как принять EULA?

Чтобы сервер начал работу, Вам нужно изменить всего один параметр

**План действий:**
1. Зайдите в панель управления сервером
2. Перейдите в раздел **Файлы**
3. Найдите в корневой папке файл `eula.txt`
4. Внутри найдите строку:
   ```properties
   eula=false
   ```
5. Измените значение `false` на `true`
   ```properties
   eula=true
   ```
6. Сохраните изменения
7. Перезапустите сервер 🔄

---

## Почему это важно?

Без принятия соглашения сервер будет выключаться сразу после запуска, а в консоли появится сообщение:
`You need to agree to the EULA in order to run the server`
